Proyectos

Busca tu anime.

Este proyecto simula una plataforma web para buscar animes, ver resultados que coincidan con el término introducido y obtener información relevante como el título, imagen, puntuación y recomendación de los usuarios. Los usuarios pueden añadir animes a una lista de favoritos y gestionarla dinámicamente. La interfaz está desarrollada con HTML y CSS, mientras que la lógica de búsqueda, filtrado y manipulación de favoritos está implementada en JavaScript puro, sin frameworks. Este proyecto demuestra mis habilidades en diseño web, manipulación del DOM, lógica de programación y experiencia de usuario.

¡Hazte con todos!!

Desarrollé un juego web interactivo inspirado en la mecánica de piedra-papel-tijera usando JavaScript, HTML y CSS puros. El usuario puede elegir su Pokémon favorito y enfrentarlo contra un rival controlado por la computadora. Implementé lógica de juego, manejo dinámico del DOM y diseño responsivo para ofrecer una experiencia visual atractiva, con animaciones y retroalimentación visual que incluye trofeos, mensajes finales y efectos de confeti. Este proyecto demuestra habilidades en manipulación de eventos, estados y estilos CSS modernos
